# Face-Recognition-based-door-lock-system-with-dual-authentication-of-RFID-card-tags-and-safety-
This project involves designing and implementing a smart door lock system that uses face recognition for primary authentication and RFID card tags for secondary verification. It also includes safety features like gas sensors and motion sensors for enhanced security. 
This project is not at all begineer friendly project. Previous knwoledge of python coding, integrating multiple sensors and thier testing and basic Linux file handeling commands are must needed to be practiced.
Since everthing can be drawn by AI, atleast basic knwoledge is required of C programming for Arduino coding, Python knwoledge for Face training, storing multiple faces in the database and recognizing when the system is active.
Please follow the basic steps to build this project.
I've used Raspberry PI 2B for the main face recognition part and Arduino board for basic sensing part. You guys can go for any of the versions of Raspberry PI, 2B and advanced versions. I recommend you to go for atleast Raspberry PI 3B+ and highly recommended Raspberry PI 4B and Raspberry PI 5. These products are expensive in the market and I will recommend to purchase a refurbished one, whcich is sufficient enough to handle complex tasks like open CV, numpy etc.
Here is the complete guide to install and setup our Raspberry PI maybe any model. https://www.raspberrypi.com/documentation/computers/getting-started.html#imager-install
Note: -  This project can also be made using ESP32 with ESPCAM module, but it will cost you more and less accurate, even the same thing goes for RPi Camera module. I'll recommend you to build the project using a USB Webcam which will cost you from 700 to 1500 making it more cost efficient.
I'll be providing with the circuit diagram as well, you can build your own circuit as well.
I'll be even providing with the code for face recognition and other sensor integration.
I;ll recommend you to keep Raspberry PI completely for face recognition system and other components will be getting controlled by arduino.
No compulsion on using Arduino, you can use any micro-controller such as ESP32, ESP8266 etc. Since I am using Gas sensor, it requires analog signals which will require an Analog to Digital converter (ADT), so for less complexity I've used Arduino.
